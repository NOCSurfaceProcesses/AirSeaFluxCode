Description
AirSeaFluxCode is a python 3.6 software designed to allow easy exploration of the different bulk formulae parameterizations and the wide range of implementation options.

Citation

Biri S., E. C. Kent, D. I. Berry, R. C. Cornes, and M. J. Yelland (2021). AirSeaFluxCode: Open-source software for calculating turbulent air-sea fluxes from meteorological parameters. Frontiers (to be submitted)
